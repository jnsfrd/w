About Scaleout

Scaleout’s mission is to enable machine learning solutions without compromising on data ownership and privacy.

Our focus

Scaleout specializes in providing a software framework that enhances applications and platforms with federated learning capabilities. This is delivered through two key products: FEDn SDK, an open-source tool allowing for extensive customization and integration, and FEDn Enterprise, which adds advanced user interface, tooling, security, and integrations for federated learning in production environments, suitable for both on-premise and virtual private cloud deployments.

Background

Scaleout is a spinoff emerging from research in applied machine learning, cloud computing, and scientific computing at Uppsala University. Our team consists of seasoned data scientists, machine learning engineers, software developers, and entrepreneurs, all with extensive experience in both academia and industry. Our backgrounds include leading roles at Google, SAAB, BCG, Autodesk, Viacom, and other major organizations, blending academic insights with practical expertise.
